Week 4 Blog Post

This week I was trying to learn more about things that I could use to spice up my webpage and blog with. I talked with a few of my other friends who are also in web development classes at different schools and and one of them told me about this library anime.js that has a lot of very modernized css tricks to make things look a little nicer on my page. The library includes many useful javascript fucntions that help with animating things specifically. I feel that this will allow me to make things like menus opening and photos scrolling on my main page automatically while the user is not scrolling through them actively. I have considered adding a splash page for when a user first navigates to the webpage that is like a loading screen. I found an example in the documentation that accomplishes the kind of thing I am looking to implement. 
            
                anime({
                    targets: '.begin-complete-demo .el',
                    translateX: 240,
                    delay: 1000,
                    easing: 'easeInOutCirc',
                    update: function(anim) {
                      progressLogEl.value = 'progress : ' + Math.round(anim.progress) + '%';
                      beginLogEl.value = 'began : ' + anim.began;
                      completeLogEl.value = 'completed : ' + anim.completed;
                    },
                    begin: function(anim) {
                      beginLogEl.value = 'began : ' + anim.began;
                    },
                    complete: function(anim) {
                      completeLogEl.value = 'completed : ' + anim.completed;
                    }
                  });

A representation of this can be found in the here. This creates a non functional yet great aesthetic addition to the webpage that I feel would serve my client for project 1 who is a graphic designer well. I want to find a way to implement this in a minimal fashion that doesnt draw too much attention away from the content on display. I also found a function that animates drawing of words in a very minimal way, this I figured I could use for the webpage header which is just my clients name. an exmaple of how to implement this was shown on the website. 

            
                anime({
                    targets: '.line-drawing-demo .lines path',
                    strokeDashoffset: [anime.setDashoffset, 0],
                    easing: 'easeInOutSine',
                    duration: 1500,
                    delay: function(el, i) { return i * 250 },
                    direction: 'alternate',
                    loop: true
                  });

I would like to learn how to implement this into my webpage in a minimal way that fits my clients vision. I also hope to be able to learn these libraries and their documentation to be able to implement them in my own code in the future.
